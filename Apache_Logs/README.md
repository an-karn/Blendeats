# Logs Parser
Simple Parsser application and printing it in terminal.

##  Getting Started

### Dependencies
* [apache_log_parser](https://github.com/rory/apache-log-parser)
* [matplotlib]


```
    pip3 install apache_log_parser 
 ```


```
    pip3 install  matplotlib 
 ```


### Run
```
    python3 logs_parser.py
```
